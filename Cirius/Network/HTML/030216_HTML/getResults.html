Forensic Databases

The Seth Faith Laboratory has developed a Altius database utilizing Python script and MySQL database. A FASTQ file will be converted to a FASTA file, and a FASTA file will be analyzed. The results include two FASTA files of matched and unmatched according to the user's selections, a matched text file including reverse and forward reads, and a second matched file in which the reverse complement of the FWD sequence is grouped with the Forward read. These can be downloaded on the page to view the results. The results of the analysis also includes the number of base pairs, the percentage of the total number of reads, the allele total number of reads, the percentage of the allele number of reads. The left and right flanking sequences are available if selected. To view the results, enter the Identification Number into the box in Altius Results.
